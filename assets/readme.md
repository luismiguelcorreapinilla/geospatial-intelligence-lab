# Assets

This directory contains the official visual resources and branding assets of **GeoSpatial Intelligence Lab (GIL)**.

These resources are shared across repositories to ensure a consistent visual identity in documentation, scientific publications, presentations, and software developed by the laboratory.

## Directory Structure

```text
assets/

├── banners/          # Repository and profile banners
├── logos/            # Official laboratory logos
├── icons/            # Icons used across documentation
├── diagrams/         # Scientific and workflow diagrams
├── figures/          # Images for README files
├── templates/        # Presentation and documentation templates
└── branding/         # Color palette, typography, and visual guidelines
```

---

## Purpose

The assets contained in this directory support the visual communication of the laboratory by providing standardized resources for:

- Repository documentation
- Scientific publications
- Technical reports
- Presentations
- Research workflows
- Public outreach

---

## Design Principles

All visual assets follow the official **GeoSpatial Intelligence Lab Brand Identity System**, ensuring consistency across all projects.

The visual identity is based on the following principles:

- Scientific rigor
- Minimalist design
- Visual consistency
- Accessibility
- Reproducibility

---

## Usage

These assets may be referenced by README files, documentation, presentations, and future laboratory websites.

Whenever possible, use the official versions stored in this directory instead of creating project-specific copies.
